# Healthcare Patient Admission Trends (EDA)

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Tasks Performed
- Loaded healthcare dataset
- Cleaned missing values
- Identified seasonal admission trends
- Calculated average length of stay
- Detected and removed outliers using the IQR rule
- Created visualizations

## Outcome
Successfully analyzed healthcare patient admission trends using Exploratory Data Analysis (EDA).
